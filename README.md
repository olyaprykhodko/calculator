# Calculator

A simple, lightweight calculator built with HTML, CSS, and JavaScript.

## Quick Start

1. Open [index.html](index.html) in your browser.
2. Use the on-screen buttons to perform calculations.

## Project Structure

- [index.html](index.html) — Markup for the calculator UI.
- [style.css](style.css) — Styling for layout and visuals.
- [script.js](script.js) — Calculator logic.
